What does it do?

This app generates quick response (QR) codes from selected M-Files objects and views, to swiftly access the same items inside M-Files mobile app. That's almost all. Small, neat and practically!

Now how do i use it?

Just select the desired object in M-Files Desktop or Web Client, right click it and choose the "Get QR-Mobilelink..." function from the context menu. That will automatically generate the QR-Code to bring that content onto your mobile device, whether it is a tablet or a smartphone. Of course M-Files mobile app needs to be installed beforehand, but it is not necessary to have it open while scanning the code - any capable QR-Code reading app will do. It is especially easy with iOS, which standard camera app now has an integrated QR-interpreter. Please note that the QR-Code is printable, if you prefer to have a physical copy of the M-Files object somewhere in reach, reusable at your fingertips.

How can we configure it?

Well there is not much to configure, since the app will do its job straightaway.
But if you'd like to set up the "Add QR-Mobile Vault Connection..." bonus function (todo: supply the full http url of your M-Files Web) or hide some of the elements of the app (like this help- or the about page), you'll need to export the app from M-Files management console, unzip it and edit the "QS_config.js" file that is part of this client extension package. The config file itself is pretty self-explaining and if you read the remarks you can easily configure it to suit your needs. Now save it and zip it again to reimport it into M-Files management console. Once M-Files clients do a logoff-logon, they'll experience the new configuration!

About this App ...
Hi there, my name is Thorsten Schwandt and I created this app.
Some of you might know me from the M-Files Partner network. Recently i was looking for a way to get objects that i opened in M-Files Desktop quickly on my mobile device and those of my co-workers without detours. So i got the idea and created this M-Files Client Extension that utilizes QR-Codes for device-spanning linkage. Since i found it to be very convenient, i would like to share this little app with you and your peers for free! All source code is included. It is tested with M-Files 2018 (Desktop and Web support) and M-Files 2015.3 (only Desktop supported). Let me know, if you like it! Enjoy!

  • Rapid step to open your customer record from the mobile
